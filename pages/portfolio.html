  • Number Game

    This simple number guessing game was my first real go at Javascript. Its not many lines of code, 20 or something? It uses a simple number generator inside of a for loop, then if statements act as flow controllers for the final output. An alert displaying the users input number, and number of guesses taken by the computer.

    This little project was actually more of a CSS challenge, because at the time I hadn't heard of flex box, so I was styling like a noob, basically.. Floats, awkward margins, absolute positioning. Although, this final version is less horrific than previous revisions. Hurrah for responsive web design!..?

    Click the link and have a go, It's mildly amusing if nothing else!
